TanStack Start (React SSR) + TanStack Router (file-based, routeTree.gen.ts auto-generated) + better-auth v1.6+ + Drizzle ORM (PostgreSQL) + pnpm.

UI: shadcn/ui components in src/components/ui/, forms via @tanstack/react-form + zod, toast via sonner.

Auth session comes from sessionAction() (server fn) → auth.api.getSession(); user is in route context via root route's beforeLoad.

**Why:** Established stack; worth knowing when adding new auth features or routes.
**How to apply:** Use pnpm (not npm/yarn). Route types auto-update on dev server start. Schema source of truth is src/db/schema/auth-schema.ts (generated by ba:generate).
